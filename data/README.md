# Data

The dataset used in this project is a synthetic remote sensing dataset provided as part of the DI725 course.

## Contents

- **images/** — True color satellite image patches
- **masks/** — Segmentation masks with 7 land cover classes: Tree, Shrub, Grass, Crop, Built-up, Barren, Water
- **captions.csv** — Per-image textual descriptions and land cover composition percentages derived from the segmentation masks.
  The CSV contains three types of captions generated by different vision-language models:
  - `hybrid_gemma3-4b` — hybrid captions combining visual and textual information
  - `text_qwen3-4b` — text-only captions generated by Qwen3-4b
  - `vision_gemma3-4b` — vision-based captions generated by Gemma3-4b
  
  Composition columns `Tree`, `Shrub`, `Grass`, `Crop`, `Built-up`, `Barren`, `Water` indicate the land cover proportion of each class per image, derived from the segmentation masks.

## Usage

This project uses the images and captions CSV. The masks are not used directly, land cover proportions computed from the masks are provided as columns in the CSV and serve as regression targets.

## Note

The dataset is not included in this repository. It is available via the course platform.
