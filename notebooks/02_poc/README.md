# Proof of Concept Notebooks

These notebooks were developed as early stage experiments to test whether incorporating textual information alongside image features improves land cover regression performance.

Different backbone architectures were evaluated (DeiT, ResNet, and SWIN) where each tested in image only and multimodal configurations on a 500 sample stratified subset of the dataset. These experiments guided backbone and fusion design decisions for the full scale experiments in Phase 2 & 3.
